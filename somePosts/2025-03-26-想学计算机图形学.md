### 学习路径：

1、 图形学基础 -> games101

2、图形学 API -> 先学 opengl，推荐《learn opengl》

3、现代图形 API -> vulkan 或者 metal，推荐 vulkan tutorial

4、工程能力，没事多做算法题（不用太难），学习各种开源项目

5、对现代 API 进行适度封装（先参考别人的做法），学习渲染技术（比如games202）并自己实现，推荐 RTR4 ，不要太早看。shader 学习，推荐 hlsl ，推荐《unityshader入门精要》
